
# Retail Analytics

## Overview

This project analyzes retail operations data to identify sales trends, regional performance, and business insights using SQL, Excel, and Power BI.

## Tools Used

* SQL
* Power BI
* Excel

## Features

* Sales trend analysis
* Regional performance analysis
* KPI dashboard creation
* Retail operations analysis
* Data cleaning and validation

## SQL Concepts Used

* Joins
* Aggregations
* Window Functions
* Subqueries

## Dashboard Insights

* Regional sales performance
* Revenue trend analysis
* Office performance tracking
* Retail KPI monitoring

## Files Included

* SQL query files
* Dashboard files
* Dataset files
* Screenshots
* Documentation and reports
